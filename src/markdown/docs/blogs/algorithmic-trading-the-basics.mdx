## Pipeline Overview

1. Ingest and clean market data
2. Generate signals from features
3. Backtest with realistic costs and slippage
4. Execute with proper risk controls

## Key Pitfalls

Avoid look-ahead bias, data snooping, and overfitting. Separate research and evaluation sets.
